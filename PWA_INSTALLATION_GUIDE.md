# The Radio Network - PWA Installation Guide

Your Radio TRN website is now a Progressive Web App (PWA) that can be installed on Android, iOS, and Windows devices!

## Features

- **Offline Support**: Browse stations even without internet
- **Fast Loading**: Cached assets for instant startup
- **Native Experience**: Works like a native app
- **Auto Updates**: Always stays up to date
- **Push Notifications**: Ready for future notification features
- **Media Controls**: Control playback from system media controls

## Installation Instructions

### Android (Chrome/Edge)

1. Open https://radiotrn.com in Chrome or Edge browser
2. Tap the menu button (3 dots) in the top right
3. Select "Install app" or "Add to Home Screen"
4. Tap "Install" in the popup
5. The app will be added to your home screen and app drawer

**Alternative method:**
- Look for the "Install" banner at the bottom of the page
- Tap "Install" directly from the banner

### iOS (Safari)

1. Open https://radiotrn.com in Safari
2. Tap the Share button (square with arrow pointing up)
3. Scroll down and tap "Add to Home Screen"
4. Edit the name if desired (RadioTRN)
5. Tap "Add" in the top right
6. The app icon will appear on your home screen

**Note:** iOS requires Safari browser for PWA installation

### Windows 10/11 (Edge/Chrome)

1. Open https://radiotrn.com in Edge or Chrome
2. Look for the install icon in the address bar (computer with down arrow)
3. Click the install icon
4. Click "Install" in the popup dialog
5. The app will open in its own window
6. Pin it to taskbar or start menu for easy access

**Alternative method:**
- Click the menu button (3 dots) in top right
- Select "Apps" → "Install The Radio Network"

### Desktop (All Browsers)

Most modern browsers support PWA installation:
- Look for an install prompt/banner
- Or check browser menu for "Install" option
- The app will run in a standalone window

## Technical Details

- **Service Worker**: Enabled for offline functionality
- **Cache Strategy**:
  - Fonts: Cache First (1 year)
  - API: Network First (5 minutes cache)
  - Static Assets: Precached
- **Icons**: Optimized for all device sizes
- **Theme**: Purple (#8b5cf6) with dark background
- **Orientation**: Portrait on mobile, flexible on desktop

## Uninstalling

### Android
- Long press the app icon
- Select "App info" or drag to "Uninstall"
- Tap "Uninstall"

### iOS
- Long press the app icon
- Select "Remove App"
- Confirm deletion

### Windows
- Right-click the app in Start Menu
- Select "Uninstall"
- Or uninstall from Settings → Apps

## Troubleshooting

**App not installing?**
- Make sure you're using HTTPS (https://radiotrn.com)
- Clear browser cache and try again
- Update your browser to the latest version
- Try a different browser (Chrome/Edge recommended)

**App not updating?**
- Close and reopen the app
- Clear app cache in browser settings
- Reinstall the app

**iOS Safari issues?**
- Make sure you're using Safari (not Chrome on iOS)
- iOS 11.3 or later required
- Some features may be limited on iOS
